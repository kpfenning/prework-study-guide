HTML

  • HTML is used to create the structure of a webpage.
  • The head element contains information about the webpage and is considered a container for metadata.
  • The body element represents the visible content shown to the user.
  • Element types are, Text, Image, Header, & List.
  • Elements nested inside other elements are called Child Elements. The element they are nested in are called Parent Elements.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations are two important components; the CSS property we want to apply and the value of the property.
  • Using * applies the rule to all elements visible on the webpage.
  • "link" connects our CSS stylesheet to our HTML.
  • "rel" attribute specifies the relationship between the current document and the linked document resource.
  • "href" attribute specifies the location.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.
  • git branch: list your branches & an * will appear next to the currently active branch.
  • git merge [branch name]: merge the specified branch's history into the current one.
